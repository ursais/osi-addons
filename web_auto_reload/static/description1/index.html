Web Auto Reload

A generic Module to reload the page, compatible with every ODOO module.

Overview

The module provides auto reload Functionality, which can be used to trigger a view reload in given interval(milliseconds) without refreshing the browser itself. We can change auto refresh time interval from edit actions option which is available in developer mode.Work for kanban/list/form view.

How To Use In Code

As it is visible in the left side screenshot, This module essentially provides us a very useful feature where we can give a auto reload value in milliseconds in the action code as per required and as per that value given in the action code, that web page will be refreshed.

How It Is Used In Spa-salon

Automatically reloads the waiting queue web page for every 5 seconds.

We can change auto refresh time interval from edit actions option which is available in developer mode.

Refresh page at a specific time interval.

Automatically reloads the chairs availability status in the front end of POS every 5 seconds.

This will ensure that as soon as a chair is available it becomes available in the front end which enables for the user to assign available chairs to new customers and new service can commence.
